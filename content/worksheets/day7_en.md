## Days 1–6 Review — The Hangul System at a Glance

Here's the complete picture of the Hangul you've learned over 7 days. No explanations — just take it all in.

### 자음 (Consonants) — 19 total

```
┌──────────────────────────────────────────────────────────────────┐
│              평음 (Plain)   격음 (Aspirated)   된소리 (Tense)    │
├──────────────────────────────────────────────────────────────────┤
│  연구개   ㄱ [k/g]          ㅋ [kʰ]             ㄲ [k͈]          │
│  치경     ㄷ [t/d]          ㅌ [tʰ]             ㄸ [t͈]          │
│  양순     ㅂ [p/b]          ㅍ [pʰ]             ㅃ [p͈]          │
│  치경마찰  ㅅ [s]            —                   ㅆ [s͈]          │
│  경구개   ㅈ [tɕ]           ㅊ [tɕʰ]            ㅉ [t͈ɕ]         │
│  후음     ㅎ [h]            —                   —               │
├──────────────────────────────────────────────────────────────────┤
│  비음     ㄴ [n]   ㅁ [m]   ㅇ [∅/ŋ]                            │
│  유음     ㄹ [ɾ/l]                                               │
└──────────────────────────────────────────────────────────────────┘
```

### 모음 (Vowels) — 21 total

```
┌──────────────────────────────────────────────────────────────────────┐
│  단모음 (10)                                                          │
│  ㅏ[a]  ㅐ[ɛ]  ㅓ[ʌ]  ㅔ[e]  ㅗ[o]  ㅚ[we]*  ㅜ[u]  ㅟ[wi]*  ㅡ[ɯ]  ㅣ[i] │
│  * Officially simple vowels, but most modern speakers pronounce      │
│    these as diphthongs (see Day 6)                                   │
├──────────────────────────────────────────────────────────────────────┤
│  이중모음 (11)                                                         │
│  y계열:  ㅑ[ja]  ㅕ[jʌ]  ㅛ[jo]  ㅠ[ju]                              │
│  w계열:  ㅘ[wa]  ㅙ[wɛ]  ㅝ[wʌ]  ㅞ[we]                              │
│  독립:   ㅢ[ɯi]                                                        │
└──────────────────────────────────────────────────────────────────────┘
```

### 7 Final-Consonant Sounds (받침) + Cluster Finals (겹받침)

```
받침 sounds:  [k̚]  [n]  [t̚]  [l]  [m]  [p̚]  [ŋ]
Common cluster finals (top 5):  ㄺ  ㄻ  ㄼ  ㄾ  ㅄ
```

That's the whole system. From here on, we use it to read actual K-pop and K-drama content.

## Real Reading 1 — K-pop Artists, Albums & Lyrics

### Reading Artist Names

How do K-pop artist names look in Hangul? Read each one out loud.

```
방탄소년단   [paŋ.tʰan.so.njʌn.dan]   BTS
              방  =  ㅂ + ㅏ + ㅇ (받침)
              탄  =  ㅌ + ㅏ + ㄴ
              소  =  ㅅ + ㅗ
              년  =  ㄴ + ㅕ + ㄴ
              단  =  ㄷ + ㅏ + ㄴ

블랙핑크   [pɯl.lɛk̚.pʰiŋ.kʰɯ]   BLACKPINK
              블  =  ㅂ + ㅡ + ㄹ
              랙  =  ㄹ + ㅐ + ㄱ
              핑  =  ㅍ + ㅣ + ㅇ
              크  =  ㅋ + ㅡ

트와이스   [tʰɯ.wa.i.sɯ]   TWICE
              트  =  ㅌ + ㅡ
              와  =  ㅇ + ㅘ
              이  =  ㅇ + ㅣ
              스  =  ㅅ + ㅡ

뉴진스   [nju.tɕin.sɯ]   NewJeans
              뉴  =  ㄴ + ㅠ
              진  =  ㅈ + ㅣ + ㄴ
              스  =  ㅅ + ㅡ

아이유   [a.i.ju]   IU
              아  =  ㅇ + ㅏ
              이  =  ㅇ + ㅣ
              유  =  ㅇ + ㅠ
```

### Album & Song Titles

```
봄날   [pom.nal]   Spring Day (BTS)
              봄  =  ㅂ + ㅗ + ㅁ
              날  =  ㄴ + ㅏ + ㄹ

불꽃   [pul.k͈ot̚]   flame / spark
              불  =  ㅂ + ㅜ + ㄹ
              꽃  =  ㄲ + ㅗ + ㅅ (받침 sound → [t̚])

쩔어   [t͈ʌ.ɾʌ]   Dope (BTS)
              쩔  =  ㅉ + ㅓ + ㄹ
              어  =  ㅇ + ㅓ

예쁘다   [je.p͈ɯ.da]   Pretty (Oh My Girl)
              예  =  ㅇ + ㅖ
              쁘  =  ㅃ + ㅡ
              다  =  ㄷ + ㅏ

밤양갱   [pam.jaŋ.gɛŋ]   밤양갱 (비비 / BIBI)
              밤  =  ㅂ + ㅏ + ㅁ   (밤 = chestnut)
              양  =  ㅇ + ㅑ + ㅇ   (양갱 = sweet jelly bar)
              갱  =  ㄱ + ㅐ + ㅇ   → 밤양갱: chestnut sweet jelly bar
```

## Real Reading 2 — Expressions You've Heard in K-dramas

You've heard these in your favourite dramas a hundred times. Now you can actually read them.

### 호칭 (Forms of Address)

```
오빠   [o.p͈a]
         Used by a female speaker toward an older male
         (brother, boyfriend, etc.)
         older brother / boyfriend (used by females)

언니   [ʌn.ni]
         Used by a female speaker toward an older female
         older sister (used by females)

형   [hjʌŋ]
         Used by a male speaker toward an older male
         older brother (used by males)

누나   [nu.na]
         Used by a male speaker toward an older female
         older sister (used by males)

선생님   [sʌn.sɛŋ.nim]   teacher
아저씨   [a.tɕʌ.s͈i]     middle-aged man / mister
```

### Emotional & Situational Expressions

```
괜찮아   [kwɛn.tɕʰa.na]   It's okay / Are you okay?
         괜  =  ㄱ + ㅙ + ㄴ
         찮  =  ㅊ + ㅏ + ㄶ (cluster final → [n])
         아  =  ㅇ + ㅏ

사랑해   [sa.ɾaŋ.hɛ]   I love you
         사  =  ㅅ + ㅏ
         랑  =  ㄹ + ㅏ + ㅇ
         해  =  ㅎ + ㅐ

보고 싶어   [po.go si.pʌ]   I miss you (lit. I want to see you)
         보고  =  ㅂ + ㅗ + ㄱ + ㅗ
         싶어  =  ㅅ + ㅣ + ㅍ (받침, resyllabified) + ㅓ

미안해   [mi.an.hɛ]   I'm sorry
         미  =  ㅁ + ㅣ
         안  =  ㅇ + ㅏ + ㄴ
         해  =  ㅎ + ㅐ

왜   [wɛ]   why
어떻게   [ʌ.t͈ʌ.kʰe]   how
```

## Writing Practice for Today

Write your favourite K-pop artist's name or a K-drama expression in Hangul yourself.

### Practice 1: Trace & Write — 사랑해

```
사랑해 [sa.ɾaŋ.hɛ]  —  I love you

Stroke order guide:
사:  ㅅ (two strokes: left diagonal + right diagonal) + ㅏ (vertical + short horizontal right)
랑:  ㄹ (five strokes) + ㅏ + ㅇ (받침, circular)
해:  ㅎ (circle + horizontal + vertical) + ㅐ (two vertical strokes)

┌──────────────────────────────────────────┐

│                                          │
```
