# Day 17. Place & Direction — The Rest of the Particles

## Day 16 Recap

What you learned in Day 16:
- Both 은/는 and 이/가 attach to the subject position — that's the common ground
- **은/는**: topic marker — "as for ___", referring back to something already mentioned, contrast. Attaches to subject, object, or complement position.
- **이/가**: subject marker — new information, exclusive emphasis, answering questions. Subject-position only.
- The 이/가 on a complement (a complementive particle) looks identical to subject 이/가 but is a different particle
- First mention → 이/가 / referring back → 은/는

Today we dig properly into the particles Day 15 only sketched out in a table.

## 1. The 에 Family — 에, 에서, 에게·한테, and Their Opposite Direction

에, 에서, 에게, 한테, 한테서, 에게서 — all branch from the same root, 에.
Which one you use depends on whether what follows is a place or a person, and which direction things move.

### 에 vs 에서 — Why English "at" Splits into Two

In English, "at" is one word.

```
I'm at school.       (being there)
I study at school.   (doing something there)
```

Both are "at school" — English doesn't distinguish.

Korean does:

```
학교에 있어요.       (being there)
학교에서 공부해요.   (doing something there)
```

The difference comes down to **what's happening at that place**.

**에 — existence, state, destination**

```
학교에 있어요.         → I'm at school.        (existence)
서울에 살아요.         → I live in Seoul.       (state)
학교에 가요.           → I go to school.        (destination)
카페에 도착했어요.     → I arrived at the café. (arrival point)
```

Verbs like 있다/없다/살다 ("exist / not exist / live") that describe a state always take 에.

**에서 — the place where an action happens**

```
학교에서 공부해요.       → I study at school.
카페에서 일해요.         → I work at the café.
공원에서 달려요.         → I run in the park.
도서관에서 책을 읽어요.  → I read at the library.
```

Side by side:

```
카페에 있어요.     → I'm at the café.    (existence — 에)
카페에서 일해요.   → I work at the café. (action — 에서)

학교에 가요.       → I go to school.     (destination — 에)
학교에서 배워요.   → I learn at school.  (action — 에서)
```

Korean splits the one English "at" into two particles. The verb decides it — a state verb takes 에, an action verb takes 에서.

### 에게 / 한테 — To a Person

English "to" works for any direction — person or place. Korean distinguishes:

```
학교에 가요.           → I go to school.           (place → 에)
친구한테 줘요.         → I give it to my friend.   (person → 한테)
```

| Particle | Register |
|------|------|
| 한테 | spoken, everyday |
| 에게 | written, formal |

```
친구한테 줬어요.       → I gave it to a friend.   (spoken)
친구에게 줬어요.       → I gave it to a friend.   (written/formal)
강아지한테 밥을 줘요.  → I give food to the dog.
선생님에게 물어봐요.   → I ask the teacher.
```

> **Wait — why doesn't 드리다 fit here?**
>
> 드리다 is the humble form of 주다 ("give") — used toward someone above you socially. "친구에게 드렸어요" sounds off, because 친구 (a friend) is your equal, not someone you'd use humble speech for.
> You'd say "할머니께 드렸어요" instead — 드리다 is reserved for elders and superiors. 에게 vs 한테 is purely a written/spoken distinction; 드리다 belongs to a completely different axis (honorifics). The full honorific system is covered later in Series 3 (Day 24–26).

### 한테서 / 에게서 — From a Person

The "서" in 에서 carries the sense of "starting from." The same 서 attached after 한테/에게 creates "from that person."

Opposite of 한테 (an outgoing direction), 한테서/에게서 marks the **incoming direction** — the source.

```
친구한테서 받았어요.   → I received it from a friend.   (spoken)
친구에게서 받았어요.   → I received it from a friend.   (written)
선생님한테서 들었어요. → I heard it from the teacher.
```

한테 vs 한테서:

```
친구한테 줬어요.       → gave TO a friend      (outgoing — to)
친구한테서 받았어요.   → received FROM a friend (incoming — from)
```

Note: for places, the source particle is 에서, not 한테서 — "서울에서 왔어요" (I came from Seoul). 한테서 is for people only.

## 2. (으)로 — Direction, Means, Material

English splits direction, means, and material across different prepositions:

```
toward Seoul     direction
by bus           means
with chopsticks  tool
made of wood     material
```

Korean uses **(으)로** for all of these. You already learned the form rule in Day 15.

```
받침 present → 으로   (지갑으로)
받침 absent  → 로     (버스로)
받침 ㄹ       → 로     (지하철로, exception)
```

### Direction

```
서울로 가요.       → I'm heading to Seoul.
왼쪽으로 가세요.   → Go left.
```

에 also marks direction, but 에 leans toward "arriving at the destination," while (으)로 leans toward "heading in that direction."

```
학교에 가요.   → I go to school.          (emphasis on arrival)
학교로 가요.   → I'm heading to school.   (emphasis on direction)
```

### Means / Tool

```
버스로 가요.           → I go by bus.
지하철로 출근해요.     → I commute by subway.
젓가락으로 먹어요.     → I eat with chopsticks.
```

### Material

```
나무로 만들었어요.     → It's made of wood.
쌀로 만든 음식이에요.  → It's food made from rice.
```

## 3. 와/과 · 하고 · (이)랑 — Together, Listing

English "and" is one word — spoken or written, it stays "and."

Korean is different — the particle itself shifts across three levels of formality.

| Particle | 받침 | Register |
|------|------|------|
| 와 | absent | written, formal |
| 과 | present | written, formal |
| 하고 | either | spoken, everyday |
| 랑 | absent | spoken, very casual |
| 이랑 | present | spoken, very casual |

```
사과와 바나나          → apple and banana   (written — no 받침 → 와)
밥과 국               → rice and soup      (written — 받침 present → 과)
사과하고 바나나        → apple and banana   (spoken)
사과랑 바나나          → apple and banana   (very casual — no 받침 → 랑)
```

```
친구와 가요.           → I go with a friend.  (written)
친구하고 가요.         → I go with a friend.  (spoken)
친구랑 가요.           → I go with a friend.  (very casual)
```

랑/이랑 is even more casual than 하고. It shows up constantly between close friends, couples, and in casual (반말) drama scenes.

```
너랑 나랑 이야기 좀 하자.
→ You and I need to talk.
```

The 받침 rule follows the same pattern as the other particles — no 받침 → 랑, 받침 present → 이랑.

```
남자친구랑 헤어졌어.    (남자친구 = no 받침)
동생이랑 싸웠어.        (동생 = 받침 present)
```

Ranked by formality: **와/과** (most formal) → **하고** (everyday) → **(이)랑** (most casual). Dramas and everyday conversation lean heavily on 하고 and (이)랑. News, textbooks, and official documents use 와/과.

## 4. 도 / 만 — Inclusion vs. Exclusion

In Day 16 you learned that 은/는 pushes out whatever particle was marking the subject/object/complement position and takes its place. 도 and 만 work exactly the same way.

### 도 — Inclusion (too / also)

```
나도 가요.             → I'm going too.
커피도 마셔요.         → I drink coffee too.
민준이도 왔어요.       → Minjun came too.
```

도 **replaces** 이/가 and 을/를.

```
내가 → 나도      (도 replaces 이/가)
커피를 → 커피도  (도 replaces 을/를)
```

Behind particles that carry real meaning — 에, 에서, 한테 — 도 **attaches after**, it doesn't take their place.

```
학교에도 가요.         → I go to school too.
친구한테도 줬어요.     → I gave to my friend too.
```

### 만 — Exclusion (only)

```
나만 알아요.           → Only I know.
커피만 마셔요.         → I only drink coffee.
오늘만 해요.           → I'm only doing it today.
```

만 also replaces 이/가·을/를, and attaches after other particles the same way.

```
학교에만 가요.         → I only go to school.
친구한테만 말했어요.   → I only told my friend.
```

> **Wait — why do 도/만 look like 은/는?**
>
> 은/는, 도, and 만 all follow the same rule: they replace the particle that marked the position (을/를·이/가), or attach after particles that carry real meaning (에·에서·한테).
> 은/는 adds contrast. 도 adds inclusion. 만 adds exclusion — same mechanism, different nuance.

## 부터 / 까지 — Starting Point, End Point (Day 15 Recap)

A quick recap of two particles Day 15 only listed in a table.

```
월요일부터            → from Monday
금요일까지            → until Friday
월요일부터 금요일까지  → from Monday to Friday
```

Used for places too, not just time.

```
서울부터 부산까지     → from Seoul to Busan
여기부터 저기까지     → from here to there
```

## Particles at a Glance

| Particle | Meaning | Form rule |
|------|-----|----------|
| 에 | existence/state, destination | always the same |
| 에서 | place where an action happens | always the same |
| 에게 / 한테 | to a person, outgoing direction | formal (에게) / casual (한테) |
| 에게서 / 한테서 | from a person, incoming direction | formal (에게서) / casual (한테서) |
| (으)로 | direction, means, material | 받침 present→으로, absent·ㄹ→로 |
| 와/과 · 하고 · (이)랑 | together, listing | 와/과: 받침 / 하고: always same / (이)랑: 받침, 3 formality tiers |
| 도 | inclusion (too, also) | replaces or attaches |
| 만 | exclusion (only) | replaces or attaches |
| 부터 / 까지 | starting point / end point | always the same |
