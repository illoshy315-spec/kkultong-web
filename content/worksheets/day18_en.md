## 1. Personal Pronoun Contractions — 이/가, 의

나, 저, and 너 are among the few irregular pronouns that change shape when they meet a particle.

### 이/가 Contraction (Subject Position)

```
나  + 가  →  내가   (나가 ✗)
저  + 가  →  제가   (저가 ✗)
너  + 가  →  네가   (구어에서는 니가로도 씀)
```

This is why you hear "내가 할게!", "제가 할게요", and "니가 뭘 알아?" constantly in dramas.

```
우리  + 가  →  우리가   (regular, no contraction)
```

우리 ("we") doesn't contract — this irregularity belongs only to 나/저/너.

It's no coincidence that all three change the same way — it's a trace of a rule shared by these three pronouns since Middle Korean. That rule just isn't productive in modern Korean anymore — it only survives fossilized in these three words. Easiest to just memorize the three pairs as a set.

### 의 Contraction (Possessive Position)

```
나  + 의  →  내    (나의 ✗, rarely used in speech)
저  + 의  →  제
너  + 의  →  네
```

```
내 가방이야.       → It's my bag.
제 잘못이 아니에요. → It's not my fault.
네 생각은 어때?     → What do you think? (lit. What's your thought?)
```

나의/저의/너의 show up occasionally in writing, but in actual conversation people almost always say 내/제/네.

> **Wait — 내가 vs 내, what's the difference?**
>
> ```
> 내가 할게.   → I'll do it.        (subject — 가 is still there)
> 내 거야.     → It's mine.          (possessive — 의 has vanished entirely)
> ```
> Both start with "내," which makes them easy to confuse. If 가 is attached, it's the subject ("내가"). If nothing is attached, it's possessive ("내").
> Same rule for 제/네 — 제가(subject) vs 제(possessive), 네가(subject) vs 네(possessive).

## 2. The Demonstrative System — 이/그/저

To understand why 이것/그것/저것 have the shapes they do, you first need to see 이/그/저.

English has two levels: this (close) / that (far).
Korean has three:

```
이  →  proximal — close to the speaker (me)              this
그  →  medial — close to the listener, or already mentioned in conversation   that
저  →  distal — far from both speaker and listener        that (over there)
```

```
이 책 (in my hand)              → this book
그 책 (the book you're holding)  → that book (near you)
저 책 (the book on that shelf)   → that book (over there)
```

그 isn't just about location — it's used constantly to point back at something already mentioned:

```
A: 어제 그 영화 봤어?
   (그 = the very thing already brought up in conversation)
```

### 이것/그것/저것 — The Contraction Chain

이것 is the formal form. In actual conversation, people almost always say 이거.

```
이것 (formal) → 이거 (spoken)
```

이거 shrinks even further when it meets a particle:

```
이거 + 가 → 이게    이게 뭐야?          → What is this?
이거 + 는 → 이건    이건 내 거야.        → This one is mine.
이거 + 를 → 이걸    이걸로 할게요.       → I'll go with this one.
```

그것/저것 follow the same chain:

```
그것 → 그거 → 그게 / 그건 / 그걸
저것 → 저거 → 저게 / 저건 / 저걸
```

```
그게 뭐야?      → What's that?
저건 뭐예요?     → What's that (over there)?
저걸 가져올게요.  → I'll bring that one over there.
```

## 3. Question Word Contractions — 누구, 무엇

Question words shrink when they meet a particle too.

```
누구 + 가 → 누가
```

```
누가 왔어요?    → Who came?
누가 그랬어?    → Who did that?
```

Same story for 무엇:

```
무엇 (formal) → 뭐 (spoken)
```

```
이게 뭐예요?    → What is this?
뭐 해?          → What are you doing?
```

뭐 shrinks one more step when a particle attaches:

```
무엇을 → 뭐를 → 뭘
```

```
뭘 먹을까?      → What should we eat?
뭘 좋아해요?    → What do you like?
```
